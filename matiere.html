Matières enseignées au 1er Semestre :

Méthodologie de la programmation (MDLP) :

Cette matière est dans le but d'acquérir des connaissances afin de programmer.
Dans ce cours on apprend à coder dans divers langage de programmation tel que Python ou C.
Projet final : Créer un Youtube Downloader en Python.

Programmation fonctionnelle (PF) :

L'objectif de ce cours est l'apprentissage de la programmation avec les algorithmes.
On apprend à utiliser le langage Racket ou le langage OCaml.

Informatique fondamentale (IF) :

Dans ce cours, on apprend toute la théorie sur les exercices pratiques de programmation.
On y apprend les notions de : logique booléenne, ensembles et fonctions, dénombrement ainsi que des bases d'algèbres.

Introduction aux moteurs de jeux (IMJ) :

L'objectif de ce cours est d'apprendre à utiliser le logiciel de moteur de jeux appelé "Godot Engine" afin de créer un jeu vidéo.
Projet final : Créer un jeu vidéo appellé Bomberman.

Création et gestion de ressources (CGR) :

Ce cours a pour objectif d'apprendre à effectuer des tâches créatifs pour le visuel du jeux vidéo crée en IMJ.
On y utilise des logiciels comme LibreSprite, Krita.

Gestion d'identité en ligne (GIL) :

Ce cours permet d'apprendre à utiliser les bonnes bases pour pratiquer et maintenir une page web personnelle.
Projet final : Créer un site web personnel en y intégrant son CV en Latex.

Pratique des machines (PDM) :

L'objectif de ce cours est d'apprendre à utiliser l'environnement Linux en apprenant à utiliser le terminal avec l'apprentissage des commandes.
